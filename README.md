# fishka

Multiplayer browser game platform built with Bun, React, and TypeScript.

## Stack

- **Runtime**: Bun
- **Frontend**: React 19 + TypeScript
- **Transport**: WebSocket (native Bun)
- **Storage**: SQLite (bun:sqlite)

## Development

```bash
bun install
bun dev
```

## Testing

```bash
bun test
```

## Linting

```bash
bun run lint        # check
bun run lint:fix    # autofix
```
